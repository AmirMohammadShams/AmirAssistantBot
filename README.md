# 🤖 Crypto & Forex Divergence Assistant

یک دستیار هوشمند، جامع و تمام‌خودکار تلگرامی برای اسکن بازارهای مالی (کریپتو، فارکس و بازار داخلی)، تشخیص واگرایی‌های قیمتی (Divergence) و ارائه خدمات هوش مصنوعی (Gemini AI).

---

## 🌟 ویژگی‌های کلیدی

### ۱. اسکنر چند-بازاری (Multi-Market)
- **کریپتو (Binance Futures):** بررسی بیش از ۳۰۰ ارز فیوچرز به صورت همزمان.
- **فارکس (Yahoo Finance):** پشتیبانی از جفت‌ارزهای جهانی (EUR/USD, GBP/USD و...).
- **بازار داخلی (Nobitex):** دریافت لحظه‌ای نرخ تتر به تومان (USDT/IRT).

### ۲. استراتژی تشخیص واگرایی دقیق (RSI Divergence)
ربات با استفاده از فرمول‌های خالص (Native) پانداس (RSI و EMA 200) بدون نیاز به پکیج‌های سنگین، بازار را بررسی می‌کند:
- 🟢 **واگرایی مثبت (Bullish):** قیمت کف پایین‌تر (Lower Low) می‌سازد اما اندیکاتور RSI کف بالاتر (Higher Low) می‌سازد. (شرط: RSI باید حتماً به محدوده اشباع فروش <= 30 رسیده باشد).
- 🔴 **واگرایی منفی (Bearish):** قیمت سقف بالاتر (Higher High) می‌سازد اما RSI سقف پایین‌تر (Lower High) می‌سازد. (شرط: RSI باید حتماً به محدوده اشباع خرید >= 70 رسیده باشد).
- 🛡 **فیلتر روند (EMA 200):** سیگنال‌هایی که در خلاف جهت شدید روند (فاصله زیاد با میانگین متحرک ۲۰۰ روزه) باشند، برای کاهش ریسک فیلتر می‌شوند.

### ۳. رسم چارت حرفه‌ای (TradingView Style)
هنگام یافتن هر سیگنال، ربات به صورت خودکار یک عکس باکیفیت از چارت تولید می‌کند که شامل موارد زیر است:
- نمودار قیمت به شکل **کندل‌استیک** (Candlesticks) با رنگ‌بندی استاندارد (سبز/قرمز).
- نمودار RSI در بخش پایینی (Subplot).
- رسم ضخیم و واضح **خط واگرایی** بین نقاط درگیر، مستقیماً روی کندل‌ها و RSI.

### ۴. سیستم اطلاع‌رسانی دوگانه (Dual Telegram Bots)
برای جلوگیری از شلوغی، سیستم پیام‌رسانی به دو بخش تقسیم شده است:
1. **ربات اصلی (دستیار شخصی):** منوی تعاملی، چت هوش مصنوعی (متن، صوت، تصویر)، مدیریت ادمین‌ها و دریافت هشدارهای سیگنال همراه با عکس چارت.
2. **ربات لاگ (Log Bot):** یک ربات کاملاً مجزا که گزارش‌های سیستمی (شروع اسکن، تعداد ارزها، خطاهای API و پایان موفقیت‌آمیز اسکن) را گزارش می‌دهد.

### ۵. زمان‌بندی خودکار (Scheduler)
سیستم بدون نیاز به دخالت کاربر، به صورت خودکار در تایم‌فریم‌های مختلف بازار را شخم می‌زند:
- هر ۱ ساعت (1h)
- هر ۴ ساعت (4h)
- روزانه (1d)
- هفتگی (1w)

## 🚀 راهنمای نصب و راه‌اندازی (Deployment)

این ربات بهینه‌سازی شده تا به راحتی و **بدون ارور** روی سرورهای ابری رایگان مثل **Render.com** اجرا شود (بدون نیاز به پکیج‌های سنگین مثل numba).

### پیش‌نیازها
1. یک ربات تلگرام در `@BotFather` بسازید (برای ربات اصلی).
2. یک ربات تلگرام دیگر بسازید (برای ربات لاگ).
3. کد API از `Google AI Studio` برای Gemini دریافت کنید.

### تنظیمات محیطی (.env)
فایل `.env` را مشابه زیر در سرور ابری خود (در بخش Environment Variables) تنظیم کنید:
```env
# تنظیمات تلگرام
TELEGRAM_TOKEN=توکن_ربات_اصلی
LOG_BOT_TOKEN=توکن_ربات_لاگ
TELEGRAM_CHAT_ID=آیدی_عددی_شما

# تنظیمات هوش مصنوعی
GEMINI_API_KEY=کلید_گوگل

# تنظیمات ترید
TESTNET=false
EXCHANGE_ID=binance
USE_EMA_FILTER=true
```
*(نکته: در سرورهای ابری خارجی نیازی به تنظیم متغیر `HTTP_PROXY` نیست).*

### اجرای محلی (Local)

python -m venv venv

pip install -r requirements.txt
